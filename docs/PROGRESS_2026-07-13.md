## Supabase 마이그레이션

- `0001` ~ `0006`: SQL Editor로 수동 적용 후 migration history 복구 완료
- `0007_product_price_history.sql`: 단가 변경 이력과 `change_product_price` RPC
- 내일 시작 시 `npx supabase migration list`로 `0007` 원격 적용 여부 확인
- `0007`이 대기 중이면 `npx supabase db push`로 적용

## 실제 발주 테스트

- 테스트 발주번호: `MO-1783931318557`
- 상품: 설도 1kg
- 예상금액: 1,000원
- 관리자 발주 접수 성공
- 최종금액 1,200원 확정 및 `상품 준비 중` 전환 성공
- 테스트 중 실중량 입력란 누락을 발견해 수정함
- 수정 후 신규 발주로 `actual_quantity` 저장 재테스트 필요
